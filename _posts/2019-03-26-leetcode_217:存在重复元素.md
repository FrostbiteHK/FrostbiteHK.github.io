## 存在重复元素

#### 题目描述：
给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

示例 1:

    输入: [1,2,3,1]
    输出: true
示例 2:

    输入: [1,2,3,4]
    输出: false
示例 3:

    输入: [1,1,1,3,3,4,3,2,4,2]
    输出: true

#### 解题思路1：先排序，然后两两比较。
```java
class Solution {
    public boolean containsDuplicate(int[] nums) {
        Arrays.sort(nums);
        for(int i = 0; i < nums.length-1; i++){
            if(nums[i] == nums[i+1])
                return true;
        }
        return false;
    }
}
```

#### 解题思路2：利用集合不能添加重复元素的特性
```java
class Solution {
    public boolean containsDuplicate(int[] nums) {
        int len = nums.length;
        HashSet<Integer> set = new HashSet<>();
        for(int i = 0; i < len; i++){
            if(!set.add(nums[i])) return true;
        }
        return false;
    }
}
```
